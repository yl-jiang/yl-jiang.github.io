Essential C++ Chapter3: Generic Programming

Standard Template Library(STL)

STL主要包括两部分：

  1. 容器
  2. 操作容器的泛型算法

容器(container)：

  1. vector; // 顺序容器(sequential container)
  2. list; // 顺序容器
  3. deque; // 顺序容器
  4. set; // 关联容器(associative container)
  5. map; // 关联容器
  6. …

顺序类容器用来维护一组排列有序，类型相同的元素。

关联容器可以让我们快速查找容器中的元素值。

vector

#include<vector>

vector以一段连续内存来存放，因此对vector进行元素的访问和提取很有效率，但是将元素插入vector和从vector中删除元素(除了对最后一个元素进行删除和在最后面的位置进行插入外)则效率很低。因为插入/删除位置右端的所有元素都必须被复制一份，并依次右移/左移。

list

#include<list>

list以双向链表(double-linked)而非连续内存来存储内容，因此可以执行前今后后退操作。list中的每一个元素都包含三个字段：value、back指针、front指针，因此在list中的任意位置进行元素的插入或者删除操作，都颇有效率。但是对list进行随机访问操作，则效率不高。

deque

#include<deque>

已连续内存存储元素，与vector不同的是，deque对于最前端(和最后端)元素的插入和删除操作效率都较高。

用以操作这些容器的所谓泛型算法(generic algorithm)：

  1. find();
  2. sort();
  3. replace();
  4. merge();
  5. …

泛型算法

#include<algorithm>

泛型算法提供了许多可用于容器类以及数组类型上的操作。这些算法称为泛型，之所以称它们为泛型，是因为它们和它们想要操作的元素类型(容器类)无关。

实现“与容器无关”的诀窍，就是不要直接在容器身上操作。而是借由一对iterator(first和last)，标示我们要进行迭代的元素范围。

泛型算法种类：

  • 搜索(search)算法
    • find() : 用于搜索无序集合中是否存在某个值，搜索范围由iterator[first, last) 标出
    • binary_search() : 用于有序集合的搜索，如果搜索到目标就返回true，否则返回false
    • count() : 返回查询的元素个数
    • search() : 比对某个容器内是否存在某个子序列，如果存在，返回一个iterator指向子序列的起始处
  • 排序(sorting)以及次序整理(ordering)算法
  • 复制(copy)、删除(delete)、替换(substitution)算法
  • 关系(relational)算法
  • 生成(generation)与质变(mutation)算法
  • 数值(numeric)算法
  • 集合(set)算法

栗子

#include<iostream>
#include<string>
#include<vector>
#include<list>

using namespace std;

bool less_than(int val_1, int val_2){
    return val_1 < val_2 ? true : false;
}

bool greater_than(int val_1, int val_2){
    return val_1 > val_2 ? true : false;
}

template<typename T>
void display(T arr){
    for (int ix = 0; (unsigned)ix < arr.size(); ++ix){
        cout << arr[ix] << " ";
    }
}


// filter_ver1()就是所谓的泛型算法：传入的函数“pred()”不关心容器的类型
vector<int> filter_ver1(vector<int> &vec, int filter_val, bool (*pred)(int, int)){
    vector<int> nvec;

    for (int ix = 0; (unsigned)ix < vec.size(); ++ix){
        if (pred(vec[ix], filter_val))
            nvec.push_back(vec[ix]);
    }
    return nvec;
}


int main(){
    int a[] = {1, 2, 3, 4, 5, 6, 7, 8};
    vector<int> vec(a, a + 8);
    int value = 4;

    display(filter_ver1(vec, value, less_than));
    cout << endl;
    display(filter_ver1(vec, value, greater_than));

    getchar();
}

泛型指针(Iterator)

底层指针行为之上的一层抽象。

  • 解决的痛点：
    • 虽然list和vector都为容器对象，我们可以对vector object进行指针的算数运算，而不能使用指针的算术运算对list中的元素进行操作。
  • Iterator具有的常规操作：
    • ++
    • *
    • ==
    • !=
    • 等等
  • 定义
    string sa[] = {"abc", "def"};
    vector<string> svec(sa, sa + 2);
    vector<string>::iterator iter = svec.begin()； // 定义一个iterator 'iter'，该iterator指向一个元素类型为string的vector的第一个元素

    const string sa_cnt[] = {"123", "456"};
    const vector<string> svec_cnt(sa_cnt, sa_cnt + 2);
    vector<string>::const_iterator iter_2 = svec_cnt.begin();  // const_vector允许我们读取vector的元素，但不允许修改

栗子

  #include<iostream>
  #include<string>
  #include<vector>
  #include<list>
  using namespace std;

  template<typename IteratorType, typename elemType>
  IteratorType find(IteratorType first, IteratorType last, const elemType &value){
      if (first == last)
          return last;
      for (; first != last; ++first){
          if (*first == value)
              return first;
      }
      return last;
  }

  int main(){
      // iterator可以适用于不同的容器

      int ia[] = {1, 2, 3, 4, 5};
      int asize = 5;
      vector<int> ivec(ia, ia + asize);
      list<int> ilist(ia, ia + asize);

      int *pia = find(ia, ia + 5, 1);
      cout << *pia << endl;

      vector<int>::iterator it;
      it = find(ivec.begin(), ivec.end(), 2);
      cout << *it << endl;

      list<int>::iterator il;
      il = find(ilist.begin(), ilist.end(), 3);
      cout << *il << endl;

      getchar();
  }

所有容器的通用操作

  • equality(==) 和 inequality(!=)，返回true或false

  • assignment(=) 赋值操作，将某个容器复制给另一个容器

  • empty() 操作，会在容器无任何内容时返回 true，否则但会 false

  • size() 返回容器内目前持有的元素个数

  • clear() 删除所有元素

  • begin() 返回一个iterator，指向容器的第一个元素

  • end() 返回一个iterator，指向容器的最后一个元素的下一位置

  • insert() 将某一或某个范围内的元素插入容器

    • iterator insert(iterator position, elemType value)：可将value插入到position之前，并返回一个iterator，指向被插入的元素；
    • void insert(iterator position, int count, elemType value)：可在position前插入count个元素，这些元素的值都为value；
      • void insert(iterator position, iterator first, iterator last)：可在position之前插入$[first, last)$所标示的个元素；
      • iterator insert(iterator position)：在position前插入元素，元素的初始值为position所属类型的默认值；

  • erase() 将容器内的单一元素或某一范围内的元素删除

    • iterator erase(iterator position)：删除position所指的元素；
    • iterator erase(iterator first, iterator last)：删除$[first, last)$范围内的元素；

  • push_back() 在容器末端插入一个元素

  • pop_back() 删除容器的最后一个元素

  • front() 访问容器的第一个元素

  • back() 访问容器的最后一个元素

注意事项

  • 指针的算术运算并不适用于list

  •   vector<int> vec[]

  • 每个标准容器都提供一个名为 begin() 得操作函数，可以返回一个iterator，指向第一个元素。另一名为 end() 得操作会返回一个iterator，指向最后一个元素得下一位置。

  • list容器实际上是一个双向链表，可以高效得进行插入和删除操作。指针的算术运算并不适用于list。

      #include<list>
  using namespace std;

  // 以下初始化方式同样适用于其它顺序容器(vector、deque)

  list<int> c0; // 空链表
  list<int> c1(3); // 建一个含有三个默认值为0的链表
  list<int> c2(5, 2);  // 建一个含有五个元素的链表，值都是2
  list<int> c3(c2);  // 建一个c2的copy链表
  list<int> c4(c1.begin(), c1.end());  // 用c1初始化c4

  int a[8] = {1, 2, 3, 4, 5, 6, 7, 8};
  list<int> c5(a, a + 8);

function object

  #include<functional>

某种class的实例对象，这类class对function call运算符做了重载操作，如此一来可使function object被当作一般函数来使用。[类似于Python中定义了__call__()方法的类的实例，该实例化的类可以当作一般函数来使用]

用途

  • function object 可作为某个函数的参数

    标准库中存在的function object

  • 算数运算

    • plus()
    • minus()
    • negate()
    • multiplies()
    • divides()
    • modules()

  • 关系运算

    • less()
    • less_equal()
    • gerater()
    • gerater_equal()
    • equal_to()
    • not_equal_to()

  • 逻辑运算

    • logical_and()
    • logical_or()
    • logical_not()

默认情况下，sort()会对传入的容器中的数据按照升序排列，这是因为默认情况下sort()使用了less_than()运算符。如果我们传入greater_than() function object，那么sort()就会返回一个按照降序排列的结果。

function object adapter

function object adapter会对function object 进行修改操作。

binder adapter

binder adapter会将function object 的参数绑定至某个特定值，使binary(二元)function object 转化为unary(一元) function object 。

  • binder adapter:bind1st：会将指定值绑定到第一个操作数

  • binder adapter:bind2nd：会将指定值绑定到第二操作数

    该操作与Python中的funtools.partial()方法十分相似。

栗子

#include<iostream>
#include<vector>
#include<algorithm>
#include<functional>

using namespace std;

vector<int> filter(vector<int> &vec, int val, const less<int> &lt){
    vector<int> nvec;
    vector<int>::iterator iter= vec.begin();

    while ((iter = find_if(iter, vec.end(), bind2nd(lt, val))) != vec.end()){
        nvec.push_back(*iter);
        iter++;
    }
    return nvec;
}

template<typename T>
void display(T arr){
    for (int ix = 0; (unsigned)ix < arr.size(); ++ix){
        cout << arr[ix] << " ";
    }
    cout << endl;
}

int main(){
    int a[] = {1, 2, 8, 3, 5, 9, 55, 20, 75};
    int b[] = {1, 2, 3, 4, 5, 6, 7, 8, 9};
    vector<int> vec_1(a, a + 9);
    vector<int> vec_2(b, b + 9);

    int value = 9;

    display(filter(vec_1, value, less<int>()));
    display(filter(vec_2, value, less<int>()));

    getchar();
}

结合template function，将上面的程序修改为更一般的形式：

#include<iostream>
#include<vector>
#include<algorithm>
#include<functional>
#include<list>

using namespace std;

template<typename inIterator, typename outIterator, typename elemType, typename funType>
outIterator filter(inIterator beg, inIterator end, outIterator out, elemType val, funType comp){
    while ((beg = find_if(beg, end, bind2nd(comp, val))) != end){
        cout << *beg << " ";
        *out++ = *beg++;  // 因为out为iterator类型，它负责保存所有符合条件的元素
    }
    cout << endl;
    return out;
}


int main(){
    // array
    int a[] = {1, 2, 8, 3, 5, 9, 55, 20, 75};
    int save_arr[9];
    filter(a, a + 9, save_arr, 9, less<int>());
    for (int ix = 0; (unsigned)ix < 9; ++ix)
        cout << save_arr[ix] << " ";

    // vector
    int b[] = {1, 2, 3, 4, 5, 6, 7, 8, 9};
    vector<int> vec(b, b + 9);
    vector<int> save_vec(9);
    filter(vec.begin(), vec.end(), save_vec.begin(), 8, less<int>());

    // list
    int c[] = {11, 12, 13, 45, 86, 23, 79, 55};
    list<int> ls(c, c + 8);
    list<int> save_ls(8);
    filter(ls.begin(), ls.end(), save_ls.begin(), 20, greater<int>());

    getchar();
}

map

#include<map>

定义

#include<map>
#include<string>

map<string, int> dicts; // 定义了一个key为string类型，value为int类型的map

map有一个名为first的member，对应于key；有一个second的member对应于value。

栗子

#include<iostream>
#include<map>
#include<string>
using namespace std;

int main(){
map<string, int> words;
string a[] = {"a", "a", "b", "c", "b", "a", "d"};

for (int ix = 0; (unsigned)ix < 7; ++ix){
    words[a[ix]]++;
}

map<string, int>::iterator iter = words.begin();
for (; iter != words.end(); ++iter)
{
    cout << "word # " << iter->first << " occur " << iter->second << " times" << endl;
}

getchar();
}

如果我们使用某个不存在的key来索引某个map对象，那么这个不存在的key会被自动加入到map中并设置其value值为0，如果只是单纯的想确认某个key值是否存在于map中要避免使用该方法，最好使用find()或者count()。

  • 需要判断某个key是否在map中，可以使用find()方法：

    #include<iostream>
#include<map>
#include<string>
using namespace std;

int main(){
map<string, int> words;
string a[] = {"a", "a", "b", "c", "b", "a", "d"};
string key = "a";

for (int ix = 0; (unsigned)ix < 7; ++ix){
    words[a[ix]]++;
}

map<string, int>::iterator iter = words.find(key);
if (iter != words.end())
    cout << "key # " << key << " in words and it's value is " << iter->second;

getchar();
}

  • 需要判断某个key是否在map中，还可以使用count()方法：

    #include<iostream>
#include<map>
#include<string>
using namespace std;

int main(){
map<string, int> words;
string a[] = {"a", "a", "b", "c", "b", "a", "d"};
string key = "r";

for (int ix = 0; (unsigned)ix < 7; ++ix){
    words[a[ix]]++;
}

if (words.count(key) != 0)
    cout << "key # " << key << " in words and it's value is " << words[key];
else
    cout << "key # " << key << " not in words";

getchar();
}

set

#include<set>

默认情况下，set会对传入的元素进行less-than排序(升序)：

#include<vector>
#include<set>
#include<iostream>
using namespace std;

int main(){
    int a[] = {10, 8, 6, 5, 2, 10, 2, -1, -5, 78};
    vector<int> vec(a, a + 10);
    set<int> iset(vec.begin(), vec.end());

    set<int>::iterator iter = iset.begin();
    for (; iter != iset.end(); ++iter)
        cout << *iter << " ";

    getchar();
}

set可以使用insert()的一元参数和双参数方法，对set对象插入单个元素和多个元素。

Iterator Inserter

#include<iterator>

三种insertion adapter

  • back_inserter(some_vec)：会以push_back()函数取代所有对some_vec进行的赋值(assignment)操作
  • inserter()：会以容器的interser()函数取代assignment操作
  • front_inserter()：会以容器的push_front()函数取代assignment运算符[显然该insertion adapter不适用于vector类型]

以上adapter并不适用于array。

栗子

#include<iostream>
#include<vector>
#include<algorithm>
#include<functional>
#include<list>
#include<iterator>

using namespace std;

template<typename inIterator, typename outIterator, typename elemType, typename funType>
outIterator filter(inIterator beg, inIterator end, outIterator out, elemType val, funType comp){
    while ((beg = find_if(beg, end, bind2nd(comp, val))) != end){
        *out++ = *beg++;  // 因为out为iterator类型，它负责保存所有符合条件的元素
    }
    return out;
}


int main(){

    // vector
    int b[] = {1, 2, 3, 4, 5, 6, 7, 8, 9};
    vector<int> vec(b, b + 9);
    vector<int> save_vec;
    // 我们传递一个未定义size大小的save_vec给filter，并将save_vec传递给back_inserter函数，那么在filter中所有对save_vec的赋值操作均会被back_push()操作所代替
    filter(vec.begin(), vec.end(), back_inserter(save_vec), 8, less<int>());
    for (int ix = 0; (unsigned)ix < save_vec.size(); ++ix)
        cout << save_vec[ix] << " ";
    cout << endl;

    // list
    int c[] = {11, 12, 13, 45, 86, 23, 79, 55};
    list<int> ls(c, c + 8);
    list<int> save_ls;
    filter(ls.begin(), ls.end(), front_inserter(save_ls), 20, greater<int>());
    // 由于list实现的特殊性，front_inserter()不适用于vector
    list<int>::iterator iter = save_ls.begin();
    for (; iter != save_ls.end(); ++iter)
        cout << *iter << " ";
    cout << endl;

    getchar();
}

iostream iterator

#include<iterator>

标准库中有供输入及输出用的iostream iterator类，分别为：istream_iterator和ostream_iterator，分别支持单一类型的元素读取和写入。

#include<iterator>
#include<iostream>
using namespace std;

// 将is定义为一个“绑定至标准输入设备”的istream_iterator
istream_iterator is(cin);

// 在定义istream_iterator时不为它指定istream对象，即代表last iterator
istream_iterator eof;

栗子

读取“input_file.txt”文件的内容，并写入到“output_file.txt”中

#include<iostream>
#include<string>
#include<algorithm>
#include<vector>
#include<iterator>
#include<fstream>
using namespace std;

int main(){

    ifstream input_file("./input_file.txt");  // 以‘r’模式打开文件
    ofstream output_file("./output_file.txt");  // 以‘w’模式打开文件

    if (!input_file || !output_file){
        cerr << "file unable open" << endl;
        return -1;
    }

    istream_iterator<string> input_stream(input_file);  // 将istream_iterator绑定到ifstream
    istream_iterator<string> end_of_file;  // 用于表示istream_iterator结束的iterator

    vector<string> text;
    copy(input_stream, end_of_file, back_inserter(text));  // 将input_stream的指定内容传入text

    sort(text.begin(), text.end());  // 对text进行排序
    ostream_iterator<string> output_stream(output_file, " ");  // 将ostream_iterator绑定到ofstream
    copy(text.begin(), text.end(), output_stream);  // 将text中指定长度的内容传给ostream_iterator，ostream_iterator再将内容传入ofstream

    getchar();
}
